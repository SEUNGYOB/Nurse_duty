# Nurse Shift Board Prototype

빠르게 확인할 수 있는 단일 파일 프로토타입입니다.

## 실행

아무 브라우저에서 `index.html`을 열어도 되지만, 링크 복사와 공유 테스트를 하려면 로컬 서버로 여는 편이 좋습니다.

```bash
python3 -m http.server 3000
```

그 다음 `http://localhost:3000`으로 접속하세요.

## 포함된 기능

- `Day / Evening / Night / OFF` 기록
- 월간 달력 보기
- 팀원 추가 및 전환
- 공유 링크 생성
- 공유 링크 불러오기
